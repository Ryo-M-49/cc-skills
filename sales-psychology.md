# セールス心理テクニック

商品やサービスを販売するコンテンツ（LP、note記事、メール等）を作成する際に、以下のテクニックを状況に応じて組み合わせる。やりすぎると胡散臭くなるので、1つのページで3〜4個を自然に織り込むのが目安。

## 1. 希少性（Scarcity）

数量や期間を限定して「今動かないと手に入らない」と感じさせる。

使い方:
- 「ベータテスト参加枠 残りXX名 / 10名限定」
- 「先着30名は永久にこの価格で利用可」
- 「3日間限定の初回価格」

注意: 嘘の限定はNG。実際に制限を設けて、到達したら本当に締め切る。

## 2. グランドファザリング（早期優遇価格）

早期に行動した人に永久割引を提供する。SaaSで特に有効。

使い方:
- 「ベータ参加者は月3,000円で永久利用可。正式リリース後は月5,000円」
- 価格表で正式価格に打ち消し線を引き、割引価格を目立たせる

効果: 「今登録しないと損」という損失回避バイアスが働く。将来の値上げに合理的な理由がある（開発が進む、機能が増える）ので自然。

## 3. 社会的証明（Social Proof）

他の人も使っている・興味を持っていることを見せる。

使い方:
- 「現在XX名が事前登録中」（リアルタイム表示）
- 「購入者の声」「導入事例」
- 「XX名が利用中」
- 統計データの引用（「退去費用に不満を持つ人は51.6%」など）

注意: 初期はデータがないので、統計データや市場データで代用する。ユーザーが増えたら実データに差し替え。

## 4. アンカリング（価格の基準点操作）

高い比較対象を先に見せて、自分の価格を安く感じさせる。

使い方:
- 「弁護士相談1時間: 10,000〜15,000円 → 本キット: 2,980円」
- 「従来ツール全機能: 月25,000円 → BuymAuto: 月3,000円」
- 正式価格→割引価格の順に見せる（打ち消し線）

効果: 最初に見た数字が基準になるので、自分の価格が相対的に安く見える。

## 5. 損失回避（Loss Aversion）

「得られるもの」より「失うもの」を強調する。人は得することより損することを2倍強く感じる。

使い方:
- 「今も誰かが、払わなくていい退去費用を払っている」
- 「知らないだけで数万円損している可能性がある」
- 「ベータ枠が埋まると通常価格でのご案内になります」

注意: 恐怖を煽りすぎない。事実に基づいた損失を淡々と伝える。

## 6. 代替コストの提示

プロに頼んだ場合のコストを具体的に示し、自分の商品の費用対効果を際立たせる。

使い方:
- 比較表形式: 「無料情報 vs 専門家 vs 本商品」の3列
- 「弁護士に頼むと1万円、このキットなら2,980円。退去費用が3,000円下がれば元が取れる」

効果: 安いか高いかではなく「何と比べて安いか」で判断させる。

## 7. 即時性（Immediacy）

「今すぐ結果が得られる」ことを強調する。

使い方:
- 「購入後すぐにメールで届きます」
- 「今日から使えます」
- 「3分で交渉文が完成」
- 「買い切り / サブスクなし / すぐ使える」

効果: 「買ったらいつ届くのか」「すぐ使えるのか」の不安を消す。

## 8. リスクリバーサル（リスクの除去）

購入のリスクを売り手側が引き受ける。

使い方:
- 返金保証がある場合: 「30日間返金保証。満足できなければ全額返金」
- 返金保証がない場合: 「メールサポート付き。使い方がわからなければ相談してください」
- 無料プランがある場合: 「まず無料で試せます」

デジタル商品で返金保証をつけるかは商品による。持ち逃げリスクがある場合は、サポートの手厚さや無料お試しで代用する。

## 9. 段階的コミットメント（Foot in the Door）

いきなり購入を求めず、小さなアクションから始めさせる。

使い方:
- 「まず無料チェックシートを受け取る」→ ステップメールで教育 → 購入提案
- LP上で「メアド登録（無料）」をメインCTA、「購入」をサブCTAにする
- 無料プランから有料プランへのアップグレード

効果: 一度小さなYesを出した人は、次の大きなYesも出しやすい。

## 10. 権威性（Authority）

信頼できる情報源や根拠を示す。

使い方:
- 「国交省ガイドラインに基づく」
- 具体的な統計データの引用（出典明記）
- メディア掲載、専門家の推薦
- 開発者自身の経験や実績

注意: ないものを作らない。公的データや事実で権威性を担保する。

## LPでの組み合わせパターン

### デジタル商品（買い切り）の場合
推奨の組み合わせ: 代替コスト + アンカリング + 損失回避 + 段階的コミットメント
```
ファーストビュー: 段階的コミットメント（無料資料のメアド登録）
問題提起: 損失回避（知らないと損する）+ 社会的証明（統計データ）
解決策: 即時性（すぐ使える）
価格: アンカリング（専門家との比較）+ 代替コスト
CTA: リスクリバーサル（サポート付き）
```

### SaaS（月額課金）の場合
推奨の組み合わせ: 希少性 + グランドファザリング + 社会的証明 + 段階的コミットメント
```
ファーストビュー: 希少性（残りXX名）+ グランドファザリング（永久価格）
問題提起: 損失回避 + 代替コスト
機能紹介: 即時性
価格: アンカリング + グランドファザリング
登録数表示: 社会的証明
CTA: 段階的コミットメント（まず無料で試す）
```
